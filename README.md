# How to run
1. `git clone --recursive https://github.com/IvanNazaruk/DifferentDimensionMe-GUI`
2. `cd DifferentDimensionMe-GUI`
3. `pip install -r requirements.txt`
4. `cd src`
5. `python main.py`
